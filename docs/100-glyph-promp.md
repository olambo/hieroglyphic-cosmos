# Prompt: Curated Top 100 Egyptian Hieroglyphs for Artistic-Cosmic Project

I am creating a project called **Hieroglyphic Cosmos**, which maps ancient Egyptian hieroglyphs to cosmic and astronomical themes. The final artwork may be displayed in museums and should be visually striking and culturally accurate. 

Please provide:

1. A list of **the 100 most important Egyptian hieroglyphs**.  
2. A **Markdown document** output, structured and numbered. Assume fixed font and pad the tables so columns line up in vim
3. Each glyph entry should include:  
   - Number (1–100)  
   - Glyph name  
   - A brief note on **why it was selected** (1–2 lines)  
   - **Gardiner code** in the last column of a table  
4. Organize the list into **logical sections** (e.g., Divine & Cosmic Symbols, Celestial & Time Symbols, Deities, Animals, Human Forms, Concepts, Royal & Ceremonial).  
5. At the top of the document, include a short summary explaining **what the list is for, the selection criteria, and reasoning** for the choices.  
6. indicate in the Gardiner column whether the Gardiner code is a variant

**Selection Guidance**:  
- Prioritize **cultural, religious, and historical significance**.  
- Choose **visually distinctive symbols** suitable for museum-quality artistic rendering.  
- Include symbols with **cosmic or thematic resonance** for a stellar mapping project.  
- Ensure the final list balances **popularly recognized glyphs** (e.g., Ankh, Eye of Horus) with **lesser-known but symbolically rich glyphs**.  

The goal is a list that is **curated, clear, and immediately usable for both scholarly and artistic purposes**, formatted in Markdown.
